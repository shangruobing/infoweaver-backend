# NFQA后端开发

#### 介绍
NFQA后端开发

#### 软件架构
后端采用Django-Rest-Framework

数据库采用MySQL+Neo4j


#### 安装教程

1.  数据库(Neo4j+MySQL)
2.  Django-Rest-Framework
3.  Vite Vue

#### 使用说明

1.  准备好通知文件(docx)格式
2.  启动Neo4j服务
3.  使用数据库端的ipynb文件将数据导入数据库(MySQL和Neo4j)
4.  启动Django服务器
5.  使用Vite打包启动Vue
